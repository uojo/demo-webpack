# demo-webpack #
开发环境：webpack + vue

## usage ##
### node ###
使用 webpack-dev-server 
```
npm run dev
```
涉及功能：
- 支持 HMR 功能，实现热更新

使用的 webpack 插件：
- case-sensitive-paths-webpack-plugin
- extract-text-webpack-plugin
- friendly-errors-webpack-plugin
- html-webpack-plugin
- webpack-bundle-analyzer
- webpack-dev-server
- webpack-merge
- webpack-cli
